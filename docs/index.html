隐私政策 (Privacy Policy) for "HTML 文件下载" Chrome Extension

生效日期：2025-05-19

感谢您使用 "HTML 文件下载" Chrome 扩展程序 (以下简称"本扩展")。我们非常重视您的个人隐私。本隐私政策旨在说明本扩展如何收集、使用和保护您的信息。

1. 信息收集与使用

本扩展的功能是帮助用户将剪贴板中的HTML代码下载为 .html 文件。为了实现这一核心功能，本扩展需要以下权限和访问：

a. clipboardRead (剪贴板读取权限):

  • 当您点击本扩展的浏览器图标或打开本扩展的操作面板时，本扩展会尝试读取您剪贴板的当前内容。
  • 此操作的目的是：
    1. 自动将剪贴板内容填充到扩展面板的文本区域中，供您预览或编辑。
    2. 检查剪贴板内容是否为有效的HTML代码。如果内容有效且是您首次复制该特定HTML内容（自上次下载后），本扩展会自动将该内容下载为一个 .html 文件。
  • 您的剪贴板内容仅在您主动与本扩展交互时（点击图标）在您的本地计算机上临时处理。

b. storage (本地存储权限):

  • 本扩展使用浏览器的本地存储功能（chrome.storage.local）临时记录一个标记，该标记代表上一次通过本扩展自动下载的HTML内容的摘要。
  • 此标记的唯一目的是防止对完全相同的HTML内容进行重复的自动下载。当您复制新的HTML内容时，此标记会被更新。
  • 此存储是临时的，并且仅存储无法识别您个人身份的内容标记。

c. downloads (文件下载权限):

  • 此权限是本扩展实现其核心和唯一目的所必需的。当剪贴板内容被识别为有效的HTML代码并满足自动下载条件，或者当您在扩展面板中点击"下载HTML"按钮时，本扩展会**且仅会**使用此权限将用户指定或确认的HTML内容，通过浏览器内置的下载功能，保存为一个 .html 文件到您的计算机默认下载位置。没有此权限，本扩展将完全无法执行其承诺的"HTML文件下载"功能。

2. 信息处理和存储

  • 所有剪贴板内容的读取和处理均在您的本地计算机上实时进行。
  • 本扩展 不会 将您的剪贴板内容、下载的HTML文件内容或任何个人身份信息上传到任何外部服务器。
  • 本扩展 不会 与任何第三方共享您的任何数据。
  • 除了上述用于防止重复下载的临时内容标记外，本扩展不存储任何用户数据。当您清除浏览器缓存或卸载本扩展时，这些临时标记通常也会被清除。

3. 数据安全

我们采取合理的措施来保护您的信息，防止未经授权的访问、使用或泄露。由于所有核心数据处理均在用户本地进行，数据泄露的风险主要取决于用户自身计算机的安全环境。

4. 隐私政策的变更

我们可能会不时更新本隐私政策。如果我们对本政策作出任何重大更改，我们将通过适当的方式（例如，在扩展的商店页面上发布通知）通知您。我们鼓励您定期查看本隐私政策以了解任何更改。
